# ASP-HR
Code for paper ASP-HR: An Adaptive Spatial Perception and Hierarchical Reasoning mechanism for Document-level Biomedical Relation Extraction.
## Datasets
- CDR: Chemical-Disease Relation dataset  
  https://biocreative.bioinformatics.udel.edu/tasks/biocreative-v/track-3-cdr/

- GDA: Gene-Disease Association dataset  
  https://bitbucket.org/alexwuhkucs/gda-extraction/src/master/

- BioRED: A rich biomedical relation extraction dataset  
  https://github.com/ncbi/BioRED
